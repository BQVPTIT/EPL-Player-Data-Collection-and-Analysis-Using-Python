2. Mục tiêu của dự án là xây dựng một quy trình hoàn chỉnh bằng Python để:

- Lưu trữ dữ liệu vào cơ sở dữ liệu

## Nội dung chính

- Trực quan hóa dữ liệu bằng biểu đồ

## Công nghệ và thư viện sử dụng
- Python
- Pandas – xử lý và phân tích dữ liệu
- BeautifulSoup – cào dữ liệu HTML
- Selenium – thu thập dữ liệu động
- SQLite – lưu trữ dữ liệu
- Matplotlib – trực quan hóa dữ liệu với các đồ thị
- Scikitlearn - phân tích, mô hình hóa
- Flask – xây dựng module
